# CommitGen Web

Web interface and landing page for CommitGen CLI tool.

## Setup

This package is ready for shadcn/ui initialization:

```bash
cd packages/web
npx shadcn@latest init
```

Follow the prompts to set up your Next.js project with shadcn/ui components.

## Development

After initialization:

```bash
npm run dev
```

## Features (Planned)

- Landing page showcasing CommitGen CLI
- Interactive demo of commit message generation
- API endpoints for web-based commit generation
- Documentation and guides
